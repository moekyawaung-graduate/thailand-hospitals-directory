# Thailand Hospitals Directory — Polished

A polished static web app for Myanmar patients searching hospitals in Thailand.

## Features
- Search hospitals by keyword
- Filter by city
- Filter by specialty
- Quick specialty buttons
- Sort A-Z / Z-A
- Favorites with localStorage
- Compare up to 3 hospitals
- Export all hospitals CSV
- Export favorites CSV
- Hospital detail modal
- Copy phone / website
- Google Maps links
- Search highlight
- Share site
- Dark mode
- Myanmar / English UI toggle
- PWA install support
- Offline fallback page
- 404 page
- robots.txt and sitemap.xml support
- Accessibility improvements
- Toast notifications
